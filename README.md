# Password Generator App

A React Native application that generates secure random passwords based on user-selected options.

## Features

- Generate random passwords
- Custom password length
- Include lowercase letters
- Include uppercase letters
- Include numbers
- Include symbols
- Form validation
- Reset functionality

## Tech Stack

- React Native
- TypeScript
- Formik
- Yup
- React Native Bouncy Checkbox

## How It Works

1. Enter password length.
2. Select character types.
3. Click **Generate Password**.
4. A random password is created and displayed.
5. Use **Reset** to clear the form.
